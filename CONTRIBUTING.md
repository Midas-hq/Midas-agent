# Contributing to Midas Agent

Thank you for your interest in contributing to Midas Agent!

## How to Contribute

### Report Bugs
- Check existing issues before opening a new one.
- Provide clear steps to reproduce, environment details, and expected/actual behavior.

### Suggest Enhancements
- Open an issue with the `enhancement` label.
- Describe the feature and why it would be useful.

### Submit Pull Requests
We accept PRs for:
- Documentation improvements
- SDK updates (Python, JavaScript)
- Example code
- Bug fixes (if you have access to the core code)

1. Fork the repository.
2. Create a feature branch.
3. Commit your changes.
4. Open a Pull Request with a clear description.

### Code of Conduct
Please read our [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions?
Open a [Discussion](https://github.com/midas-agent/midas-agent/discussions) or email support@midas-agent.com.
